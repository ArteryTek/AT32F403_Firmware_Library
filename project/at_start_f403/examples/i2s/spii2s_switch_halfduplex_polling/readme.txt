  this demo is based on the at-start board, in this demo, shows in halfduplex
  mode how to use polling transfer data. use the mode switch to realize spi and
  i2s communication.
  the pins connection as follow:
    i2s2/spi2           i2s1/spi1
  - pb12      <--->     pa15
  - pb13      <--->     pb3
  - pb15      <--->     pb5
